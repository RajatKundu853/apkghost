Packages : metasploit-framework apktool default-jdk aapt apksigner apache2

## How To Use apkghost

sudo apt update
sudo apt install git
git clone https://github.com/RajatKundu853/apkghost.git
cd apkghost
sudo chmod +x apkghost.sh
